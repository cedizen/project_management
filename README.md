Global project monitoring dashboard (Power BI)

This project was developped for a company aiming to monitor and follow all their ongoing projects worldwide through a centralized dashboard. 
The goal was to provide real-time visibility and alerting based on three key performance indicators :
- Costs : tracking planned and real expenses
- Deliverables : monitoring process and scope completion
- Duration : comparing actual vs planned timelines

A 15% deviation threshold was defined for each indicator. Whenever one of those indicators exceeded the threshold, an automatic alert was triggered 
flag potential risks

The Power BI dashboard was designed : 
- to consolidate projects data from multiples regions and coutnry into a one interactive view
- Display KPIs at global, regional and project levels
- Highlight projects with active alerts for rapid management action
